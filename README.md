# LuxeTravel Hub

A premium, standalone frontend application for luxury travel experiences. Built with React, TypeScript, Tailwind CSS, and shadcn/ui.

## Features

- **Premium UI/UX**: Modern glassmorphism and smooth animations.
- **Dynamic Sections**: Hero, Destination Cards, Tour Packages, Itineraries, and Reviews.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Standalone**: No external backend dependencies (uses local state/mock data).

## Tech Stack

- **Framework**: React 18
- **Styling**: Tailwind CSS
- **Components**: shadcn/ui
- **Icons**: Lucide React
- **Build Tool**: Vite

## Getting Started

1. **Install Dependencies**:
   ```sh
   npm install
   ```

2. **Run Development Server**:
   ```sh
   npm run dev
   ```

3. **Build for Production**:
   ```sh
   npm run build
   ```

## Folder Structure

- `src/components`: Reusable UI components.
- `src/pages`: Main page layouts.
- `src/hooks`: Custom React hooks for animations and state.
- `src/lib`: Utility functions and configuration.
- `public`: Static assets and images.
